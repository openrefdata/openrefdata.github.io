What is Open Reference Data?

Open Bond Reference data provides the financial industry with a free open source security master dataset. Our solution is a freely available consistent security master data set that can be used throughout an organization.
Today, we source our data from government announcements and public filings (e.g. bond prospectuses). We make all our data available for delivery via our standard download functionality.

What is Security Master Data?

Security Master Data is one of the essential data sets needed by all financial institutions. The Security Master Dataset contains a set of fields that describe the security.
